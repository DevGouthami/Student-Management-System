# **Student Management System (Java)**

This is a simple **Java console project** that allows you to manage student details.
You can add, view, search, update, and delete students using a menu-based system.

---

## **Features**

* Add Student
* View All Students
* Search Student
* Delete Student
* Update Student

---

## **Concepts Used**

* Classes & Objects
* ArrayList
* Methods
* Scanner (user input)
* Basic Java OOP

---

## **How to Run**

1. Open the project in VS Code / IntelliJ / any Java IDE
2. Compile the code:

   ```
   javac Main.java
   ```
3. Run the program:

   ```
   java Main
   ```
